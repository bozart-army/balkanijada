# Balkanijada Backend (NestJS)

This is the backend API for Balkanijada.
It provides REST endpoints for:

- Places
- Events
- Stories
- Users
- Organizers
- Translations (i18n)
- AI workflows (later)

Tech stack:
- NestJS
- TypeScript
- Prisma ORM
- PostgreSQL
- JWT Authentication

Runbook and environment setup will be added later.
